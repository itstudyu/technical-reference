HTTP Methods

HTTP methods (also called verbs) indicate the desired action to be performed on a resource.

GET

Purpose: Retrieve data from server

  • Idempotent and safe
  • Data sent in URL parameters
  • Can be cached
  • Should not modify server state
GET /api/users/123 HTTP/1.1
Host: api.example.com

POST

Purpose: Submit data to server

  • Not idempotent
  • Data sent in request body
  • Used for creating resources
  • Can modify server state
POST /api/users HTTP/1.1
Content-Type: application/json

{"name": "John", "email": "john@example.com"}

PUT

Purpose: Update/replace entire resource

  • Idempotent
  • Replaces entire resource
  • Creates resource if it doesn't exist
  • Data sent in request body
PUT /api/users/123 HTTP/1.1
Content-Type: application/json

{"name": "John Smith", "email": "john.smith@example.com"}

PATCH

Purpose: Partially update resource

  • May or may not be idempotent
  • Updates specific fields only
  • More efficient than PUT
  • Data sent in request body
PATCH /api/users/123 HTTP/1.1
Content-Type: application/json

{"email": "newemail@example.com"}

DELETE

Purpose: Remove resource from server

  • Idempotent
  • Removes specified resource
  • Usually no request body
  • May return content about deletion
DELETE /api/users/123 HTTP/1.1
Host: api.example.com

HEAD

Purpose: Get headers without body

  • Same as GET but no response body
  • Used to check resource existence
  • Check if resource modified
  • Get metadata about resource
HEAD /api/users/123 HTTP/1.1
Host: api.example.com

HTTP Status Codes

HTTP status codes indicate whether a specific HTTP request has been successfully completed.

1xx - Informational

Request received, continuing process

100 Continue: Server received initial part of request
101 Switching Protocols: Server switching to different protocol

2xx - Success

Request was successfully received, understood, and accepted

200 OK: Request succeeded
201 Created: Request succeeded and new resource created
204 No Content: Request succeeded but no content to return

3xx - Redirection

Further action needs to be taken to complete request

301 Moved Permanently: Resource permanently moved to new URL
302 Found: Resource temporarily moved
304 Not Modified: Resource not modified since last request

4xx - Client Error

Request contains bad syntax or cannot be fulfilled

400 Bad Request: Invalid request syntax
401 Unauthorized: Authentication required
403 Forbidden: Access denied
404 Not Found: Resource not found
429 Too Many Requests: Rate limit exceeded

5xx - Server Error

Server failed to fulfill valid request

500 Internal Server Error: Generic server error
502 Bad Gateway: Invalid response from upstream server
503 Service Unavailable: Server temporarily unavailable
504 Gateway Timeout: Upstream server timeout

üêç Python HTTP Client

import requests
import json

class HTTPClient:
    def __init__(self, base_url, headers=None):
        self.base_url = base_url.rstrip('/')
        self.session = requests.Session()
        if headers:
            self.session.headers.update(headers)
    
    def get(self, endpoint, params=None):
        """Make GET request"""
        url = f"{self.base_url}/{endpoint.lstrip('/')}"
        response = self.session.get(url, params=params)
        response.raise_for_status()
        return response.json() if response.content else None
    
    def post(self, endpoint, data=None, json_data=None):
        """Make POST request"""
        url = f"{self.base_url}/{endpoint.lstrip('/')}"
        if json_data:
            response = self.session.post(url, json=json_data)
        else:
            response = self.session.post(url, data=data)
        response.raise_for_status()
        return response.json() if response.content else None
    
    def put(self, endpoint, data=None, json_data=None):
        """Make PUT request"""
        url = f"{self.base_url}/{endpoint.lstrip('/')}"
        if json_data:
            response = self.session.put(url, json=json_data)
        else:
            response = self.session.put(url, data=data)
        response.raise_for_status()
        return response.json() if response.content else None
    
    def delete(self, endpoint):
        """Make DELETE request"""
        url = f"{self.base_url}/{endpoint.lstrip('/')}"
        response = self.session.delete(url)
        response.raise_for_status()
        return response.json() if response.content else None

üü® JavaScript HTTP Client

class HTTPClient {
    constructor(baseURL, defaultHeaders = {}) {
        this.baseURL = baseURL.replace(/\/$/, '');
        this.defaultHeaders = {
            'Content-Type': 'application/json',
            ...defaultHeaders
        };
    }

    async request(method, endpoint, options = {}) {
        const url = `${this.baseURL}/${endpoint.replace(/^\//, '')}`;
        
        const config = {
            method: method.toUpperCase(),
            headers: {
                ...this.defaultHeaders,
                ...options.headers
            }
        };

        // Add body for POST, PUT, PATCH requests
        if (['POST', 'PUT', 'PATCH'].includes(config.method) && options.data) {
            if (typeof options.data === 'object') {
                config.body = JSON.stringify(options.data);
            } else {
                config.body = options.data;
            }
        }

        // Add query parameters
        if (options.params) {
            const searchParams = new URLSearchParams(options.params);
            url += `?${searchParams.toString()}`;
        }

        try {
            const response = await fetch(url, config);
            
            if (!response.ok) {
                const error = new Error(`HTTP Error: ${response.status} ${response.statusText}`);
                error.status = response.status;
                error.response = response;
                throw error;
            }

            // Handle different content types
            const contentType = response.headers.get('content-type');
            if (contentType && contentType.includes('application/json')) {
                return await response.json();
            } else if (contentType && contentType.includes('text/')) {
                return await response.text();
            } else {
                return await response.blob();
            }
        } catch (error) {
            console.error(`Request failed: ${method} ${url}`, error);
            throw error;
        }
    }

    get(endpoint, params = {}) {
        return this.request('GET', endpoint, { params });
    }

    post(endpoint, data, headers = {}) {
        return this.request('POST', endpoint, { data, headers });
    }

    put(endpoint, data, headers = {}) {
        return this.request('PUT', endpoint, { data, headers });
    }

    patch(endpoint, data, headers = {}) {
        return this.request('PATCH', endpoint, { data, headers });
    }

    delete(endpoint, headers = {}) {
        return this.request('DELETE', endpoint, { headers });
    }
}

üîí HTTPS Server (Node.js)

const https = require('https');
const http = require('http');
const fs = require('fs');
const path = require('path');

class HTTPSServer {
    constructor(options = {}) {
        this.port = options.port || 443;
        this.httpPort = options.httpPort || 80;
        this.certPath = options.certPath || './ssl/cert.pem';
        this.keyPath = options.keyPath || './ssl/key.pem';
        this.redirectHttp = options.redirectHttp !== false;
        
        this.routes = new Map();
        this.middleware = [];
    }

    use(middleware) {
        this.middleware.push(middleware);
    }

    get(path, handler) {
        this.addRoute('GET', path, handler);
    }

    post(path, handler) {
        this.addRoute('POST', path, handler);
    }

    put(path, handler) {
        this.addRoute('PUT', path, handler);
    }

    delete(path, handler) {
        this.addRoute('DELETE', path, handler);
    }

    addRoute(method, path, handler) {
        const key = `${method}:${path}`;
        this.routes.set(key, handler);
    }

    async handleRequest(req, res) {
        try {
            // Add security headers
            this.setSecurityHeaders(res);

            // Apply middleware
            for (const middleware of this.middleware) {
                await middleware(req, res);
            }

            // Parse request body for POST/PUT requests
            if (['POST', 'PUT', 'PATCH'].includes(req.method)) {
                req.body = await this.parseBody(req);
            }

            // Find and execute route handler
            const key = `${req.method}:${req.url.split('?')[0]}`;
            const handler = this.routes.get(key);

            if (handler) {
                await handler(req, res);
            } else {
                this.sendNotFound(res);
            }
        } catch (error) {
            console.error('Request error:', error);
            this.sendError(res, error);
        }
    }

    setSecurityHeaders(res) {
        res.setHeader('Strict-Transport-Security', 'max-age=31536000; includeSubDomains');
        res.setHeader('X-Content-Type-Options', 'nosniff');
        res.setHeader('X-Frame-Options', 'DENY');
        res.setHeader('X-XSS-Protection', '1; mode=block');
        res.setHeader('Content-Security-Policy', "default-src 'self'");
        res.setHeader('Referrer-Policy', 'strict-origin-when-cross-origin');
    }

    async parseBody(req) {
        return new Promise((resolve, reject) => {
            let body = '';
            req.on('data', chunk => {
                body += chunk.toString();
            });
            req.on('end', () => {
                try {
                    const contentType = req.headers['content-type'] || '';
                    if (contentType.includes('application/json')) {
                        resolve(JSON.parse(body));
                    } else {
                        resolve(body);
                    }
                } catch (error) {
                    reject(error);
                }
            });
        });
    }

    sendJSON(res, data, status = 200) {
        res.writeHead(status, { 'Content-Type': 'application/json' });
        res.end(JSON.stringify(data));
    }

    sendNotFound(res) {
        res.writeHead(404, { 'Content-Type': 'application/json' });
        res.end(JSON.stringify({ error: 'Not Found' }));
    }

    sendError(res, error, status = 500) {
        res.writeHead(status, { 'Content-Type': 'application/json' });
        res.end(JSON.stringify({ 
            error: error.message || 'Internal Server Error' 
        }));
    }

    start() {
        try {
            // Load SSL certificates
            const httpsOptions = {
                key: fs.readFileSync(this.keyPath),
                cert: fs.readFileSync(this.certPath)
            };

            // Create HTTPS server
            const httpsServer = https.createServer(httpsOptions, (req, res) => {
                this.handleRequest(req, res);
            });

            httpsServer.listen(this.port, () => {
                console.log(`HTTPS Server running on port ${this.port}`);
            });

            // Create HTTP server for redirects (optional)
            if (this.redirectHttp) {
                const httpServer = http.createServer((req, res) => {
                    const host = req.headers.host.split(':')[0];
                    const httpsUrl = `https://${host}${req.url}`;
                    
                    res.writeHead(301, {
                        'Location': httpsUrl,
                        'Strict-Transport-Security': 'max-age=31536000'
                    });
                    res.end();
                });

                httpServer.listen(this.httpPort, () => {
                    console.log(`HTTP Server redirecting on port ${this.httpPort}`);
                });
            }

        } catch (error) {
            console.error('Failed to start server:', error);
        }
    }
}

// Usage example
const server = new HTTPSServer({
    port: 443,
    httpPort: 80,
    certPath: './ssl/fullchain.pem',
    keyPath: './ssl/privkey.pem'
});

// Add middleware
server.use(async (req, res) => {
    console.log(`${new Date().toISOString()} - ${req.method} ${req.url}`);
});

// Add routes
server.get('/', (req, res) => {
    server.sendJSON(res, { message: 'Hello HTTPS World!' });
});

server.get('/api/users', (req, res) => {
    server.sendJSON(res, [
        { id: 1, name: 'Alice', email: 'alice@example.com' },
        { id: 2, name: 'Bob', email: 'bob@example.com' }
    ]);
});

server.post('/api/users', (req, res) => {
    const newUser = { id: Date.now(), ...req.body };
    server.sendJSON(res, newUser, 201);
});

// Start the server

Best Practices

üîí Always Use HTTPS

Use HTTPS for all websites, not just those handling sensitive data. Implement HSTS (HTTP Strict Transport Security) to force HTTPS connections.

üìä Proper Status Codes

Use appropriate HTTP status codes. Don't return 200 OK for errors or 500 for client mistakes. Each status code has a specific meaning.

üè∑Ô∏è Meaningful Headers

Include relevant headers like Content-Type, Cache-Control, and security headers. Use custom headers sparingly and follow the X- prefix convention.

‚ö° Performance Optimization

Enable compression (gzip/brotli), use caching effectively, minimize header size, and implement HTTP/2 for better performance.

üõ°Ô∏è Security Headers

Implement security headers like CSP, HSTS, X-Frame-Options, and X-Content-Type-Options to protect against common attacks.

üîÑ Idempotency

Ensure GET, PUT, and DELETE operations are idempotent. The same request should produce the same result when repeated.

‚ö†Ô∏è Common Mistakes to Avoid

  • Mixed Content: Don't load HTTP resources on HTTPS pages
  • Sensitive Data in URLs: Never put passwords or tokens in GET parameters
  • Weak Certificates: Use strong encryption and keep certificates updated
  • Ignoring CORS: Properly configure CORS for cross-origin requests
  • Poor Error Handling: Don't expose internal server details in errors
  • Caching Sensitive Data: Use proper cache headers for sensitive content
